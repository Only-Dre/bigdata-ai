# Objetivo Exercício 1 - Titanic
---
O objetivo é prever quais passageiros do Titanic sobreviveriam, usando os arquivos de dados fornecidos.
O aluno deve limpar e preparar os dados, transformar variáveis categóricas, separar X e y, e treinar quatro modelos de classificação.
Depois, deve avaliar cada modelo usando acurácia, precisão, recall e F1-Score e escolher o melhor com base nos resultados.

# Objetivo Exercício 2 - Tsunami
O objetivo é prever se um terremoto gera ou não um tsunami, usando atributos como magnitude, profundidade e localização.
O aluno deve explorar os dados, preparar X e y, dividir em treino e teste, treinar quatro modelos e avaliar seu desempenho com métricas de classificação.
Como falhas podem ser críticas, a métrica mais importante é o recall.
